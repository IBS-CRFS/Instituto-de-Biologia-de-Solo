## Por que CC BY 4.0?

A licença CC BY 4.0 é apropriada para datasets com valor agregado porque:

1. **Reconhece o trabalho**: Garante atribuição aos criadores
2. **Permite reutilização ampla**: Maximiza o impacto dos dados
3. **Flexível**: Permite uso comercial e modificações
4. **Padrão internacional**: Amplamente reconhecida e adotada
5. **Compatível**: Pode ser integrada com outros dados CC BY
6. **Rastreável**: Facilita o rastreamento de uso e impacto

## Compatibilidade com CC0

Dados sob CC0 podem ser incorporados em derivações CC BY 4.0, mas não o inverso. Se você combinar:

- **CC0 + CC BY → CC BY**: O resultado deve ser CC BY
- **CC BY + CC BY → CC BY**: Compatível
- **CC BY → CC0**: Não permitido sem permissão adicional do criador original

## Perguntas Frequentes

### 1. Posso usar os dados comercialmente?
**Sim.** CC BY 4.0 permite uso comercial com atribuição apropriada.

### 2. Posso modificar os dados?
**Sim.** Você pode adaptar, transformar e construir sobre os dados, desde que dê crédito e indique mudanças.

### 3. Preciso disponibilizar minhas modificações sob a mesma licença?
**Não.** CC BY 4.0 não é "copyleft" - você pode licenciar suas derivações sob termos diferentes, mas deve manter a atribuição aos dados originais.

### 4. Posso usar em publicações acadêmicas?
**Sim.** A atribuição via citação normal atende aos requisitos da licença.

### 5. E se eu usar apenas uma pequena parte dos dados?
**A atribuição ainda é necessária.** O tamanho da porção usada não elimina o requisito de crédito.
