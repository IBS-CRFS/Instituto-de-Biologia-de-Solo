## Por que CC0?

A licença CC0 é recomendada para dados científicos porque:

1. **Maximiza reutilização**: Elimina barreiras legais ao uso dos dados
2. **Facilita integração**: Permite combinar dados de múltiplas fontes
3. **Conformidade internacional**: Reconhecida globalmente
4. **Promove ciência aberta**: Alinha-se com princípios FAIR e de dados abertos
5. **Evita ambiguidades**: Clareza sobre direitos e permissões

## Atribuição (opcional mas recomendada)

Embora não seja legalmente obrigatório sob CC0, a comunidade científica valoriza a atribuição apropriada. Ao usar dados sob CC0 deste repositório, recomendamos:
