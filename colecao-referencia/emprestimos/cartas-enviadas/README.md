# Cartas Enviadas

Esta pasta armazena todas as **correspondências oficiais enviadas** pelo Instituto de Biologia do Solo relacionadas a empréstimos da coleção.

## Organização

As cartas são organizadas por tipo em subpastas:

```
cartas-enviadas/
├── aprovacao/           # Cartas aprovando empréstimos
├── negacao/            # Cartas negando solicitações
├── cobranca/           # Lembretes de devolução
├── confirmacao/        # Confirmações de recebimento
└── agradecimento/      # Agradecimentos por devoluções
```

## Nomenclatura de Arquivos

Siga o padrão:
```
YYYY-MM-DD_tipo_instituicao_pesquisador.pdf
```

**Exemplos**:
- `2024-01-15_aprovacao_usp_silva.pdf`
- `2024-02-20_cobranca_unicamp_santos.pdf`
- `2024-03-10_confirmacao_ufrj_oliveira.pdf`

## Conteúdo Obrigatório

Toda carta enviada deve incluir:
- ✅ Número de referência do empréstimo
- ✅ Data de envio
- ✅ Instituição destinatária
- ✅ Pesquisador responsável
- ✅ Resumo do conteúdo
- ✅ Assinatura do curador

## Templates

Use os templates oficiais disponíveis em:
`/templates-institucionais/cartas/`

## Digitalização

- Todas as cartas físicas devem ser digitalizadas
- Resolução mínima: 300 DPI
- Formato: PDF
- Armazenar original físico em arquivo

## Backup

- Backup automático mensal
- Cópias mantidas em servidor seguro
- Registro no sistema de empréstimos (`/emprestimos/registros/`)

Ver documentação completa em `/colecao-referencia/emprestimos/README.md`
