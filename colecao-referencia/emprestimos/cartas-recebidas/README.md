# Cartas Recebidas

Esta pasta armazena todas as **correspondências recebidas** de pesquisadores e instituições relacionadas a empréstimos da coleção.

## Organização

As cartas são organizadas por tipo em subpastas:

```
cartas-recebidas/
├── solicitacoes/        # Solicitações de empréstimo
├── renovacoes/          # Pedidos de renovação de prazo
├── devolucoes/          # Avisos de devolução
└── agradecimentos/      # Cartas de agradecimento
```

## Nomenclatura de Arquivos

Siga o padrão:
```
YYYY-MM-DD_tipo_instituicao_pesquisador.pdf
```

**Exemplos**:
- `2024-01-10_solicitacao_usp_silva.pdf`
- `2024-04-15_renovacao_unicamp_santos.pdf`
- `2024-06-20_devolucao_ufrj_oliveira.pdf`

## Processamento

Ao receber uma carta:

1. ✅ Digitalizar (se recebida fisicamente)
2. ✅ Salvar com nomenclatura padronizada
3. ✅ Criar entrada no registro de empréstimos
4. ✅ Preparar e enviar resposta
5. ✅ Arquivar original físico

## Análise de Solicitações

Para cada solicitação, verificar:
- Qualificação do pesquisador
- Justificativa científica
- Disponibilidade do material
- Condições de armazenamento
- Prazo solicitado

## Tempo de Resposta

- **Confirmação de recebimento**: 48 horas
- **Resposta sobre aprovação**: 7-14 dias
