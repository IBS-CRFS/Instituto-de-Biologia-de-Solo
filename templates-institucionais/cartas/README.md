# Templates de Cartas Oficiais

Modelos de **cartas oficiais** do Instituto de Biologia do Solo.

## Estrutura

```
cartas/
├── timbradas/          # Papel timbrado IBS
├── solicitacoes/       # Templates de solicitação
├── respostas/          # Templates de resposta
└── administrativas/    # Cartas administrativas
```

## Tipos de Templates

### 1. Timbradas
- Papel timbrado institucional
- Formatos: DOCX, ODT, PDF
- Com logo, endereço, contatos

### 2. Solicitações
- Solicitação de empréstimo
- Solicitação de visita
- Solicitação de doação

### 3. Respostas
- Aprovação de empréstimo
- Negação de empréstimo
- Confirmação de recebimento
- Cobrança de devolução

### 4. Administrativas
- Agradecimento
- Convite para eventos
- Certificado de depósito
- Relatório de curadoria

## Como Usar

1. Abra o template apropriado
2. Preencha os campos entre [colchetes]
3. Revise cuidadosamente
4. Salve com nome descritivo
5. Converta para PDF para envio oficial
6. Arquive cópia na pasta apropriada

## Campos Padrão

Todos os templates têm campos para:
- `[DATA]`
- `[NOME_DESTINATARIO]`
- `[INSTITUICAO_DESTINATARIO]`
- `[ENDERECO_DESTINATARIO]`
- `[NUMERO_REFERENCIA]`
- `[NOME_CURADOR]`
- `[ASSINATURA_CURADOR]`

## Formatos

- **DOCX**: Microsoft Word (edição fácil)
- **ODT**: LibreOffice (software livre)
- **PDF**: Visualização/impressão

## Personalização

Não modifique:
- ❌ Logo do IBS
- ❌ Cores institucionais
- ❌ Layout do cabeçalho/rodapé
- ❌ Fonte oficial

Pode modificar:
- ✅ Conteúdo do corpo da carta
- ✅ Parágrafos específicos
- ✅ Listas e tabelas no corpo

Ver documentação completa em `/templates-institucionais/README.md`
