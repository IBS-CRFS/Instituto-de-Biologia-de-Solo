# Papel Timbrado

Templates de **papel timbrado oficial** do Instituto de Biologia do Solo.

## Arquivos Disponíveis

| Arquivo | Formato | Software | Uso |
|---------|---------|----------|-----|
| `papel_timbrado_ibs.docx` | Word | Microsoft Office | Edição rápida |
| `papel_timbrado_ibs.odt` | ODT | LibreOffice | Software livre |
| `papel_timbrado_ibs.pdf` | PDF | Visualizador | Impressão/visualização |

## Elementos do Timbrado

### Cabeçalho (topo)
- Logotipo IBS (esquerda)
- Nome completo da instituição
- Linha decorativa

### Rodapé (parte inferior)
- Endereço completo
- Telefone e email
- Website institucional
- CNPJ (se aplicável)

## Como Usar

1. **Abra** o arquivo DOCX ou ODT
2. **Digite** o conteúdo da carta no corpo
3. **Não modifique** cabeçalho nem rodapé
4. **Salve** com nome descritivo
5. **Exporte** para PDF para envio oficial

## Configurações

- **Papel**: A4 (210 x 297 mm)
- **Margens**:
  - Superior: 3 cm (espaço para cabeçalho)
  - Inferior: 2 cm (espaço para rodapé)
  - Esquerda/Direita: 2,5 cm
- **Fonte**: Times New Roman 12pt
- **Espaçamento**: 1,5 linhas

## Área de Texto

- Inicie a escrita após o cabeçalho
- Mantenha espaço adequado das margens
- Não invada área do rodapé

## Impressão

- Qualidade: Alta (300 DPI mínimo)
- Papel: Branco, gramatura 75-90g
- Impressora: Laser (melhor qualidade)
- Cores: Colorida (logo deve estar em cores)

## Digitalização

Se impresso e assinado fisicamente:
- Digitalizar em 300 DPI
- Salvar como PDF
- Nomear adequadamente
- Arquivar em pasta apropriada

Para cartas relacionadas a empréstimos, arquivar em:
`/colecao-referencia/emprestimos/cartas-enviadas/`

Ver mais templates em: `/templates-institucionais/cartas/`
