# Logotipos do IBS

Esta pasta contém todos os **logotipos oficiais** do Instituto de Biologia do Solo em múltiplos formatos e variações.

## Estrutura

```
logotipos/
├── principal/      # Logo principal em várias resoluções
├── variantes/      # Versões alternativas
└── uso/           # Manual de identidade visual
```

## Formatos Disponíveis

### Principal
- `ibs_logo.svg` - Vetorial (uso geral)
- `ibs_logo_hires.png` - Alta resolução (300 DPI)
- `ibs_logo_web.png` - Web (150 DPI)
- `ibs_logo_thumb.png` - Miniatura (72 DPI)
- `ibs_logo.eps` - Gráfica profissional
- `ibs_logo.pdf` - Documentos

### Variantes
- Monocromático (preto/branco)
- Simplificado
- Ícone apenas
- Cores alternativas

## Manual de Identidade

O arquivo `/uso/manual_identidade_visual.pdf` contém:
- Cores oficiais (RGB, CMYK, Pantone, HEX)
- Espaçamentos mínimos
- Tamanhos mínimos
- Fundos permitidos
- Usos incorretos

## Cores Oficiais

**Verde IBS (primária)**:
- RGB: 34, 139, 34
- HEX: #228B22
- CMYK: 76, 0, 76, 45

**Terra (secundária)**:
- RGB: 139, 90, 43
- HEX: #8B5A2B
- CMYK: 0, 35, 69, 45
