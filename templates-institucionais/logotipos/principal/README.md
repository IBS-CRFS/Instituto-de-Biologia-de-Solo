# Logo Principal

Logotipo **principal** do Instituto de Biologia do Solo em múltiplas resoluções e formatos.

## Arquivos

| Arquivo | Formato | Resolução | Uso Recomendado |
|---------|---------|-----------|-----------------|
| `ibs_logo.svg` | SVG vetorial | Escalável | Design, impressão gráfica |
| `ibs_logo_hires.png` | PNG | 300 DPI | Impressão de alta qualidade |
| `ibs_logo_web.png` | PNG | 150 DPI | Website, apresentações |
| `ibs_logo_thumb.png` | PNG | 72 DPI | Ícones, miniaturas |
| `ibs_logo.eps` | EPS | Vetorial | Gráfica profissional |
| `ibs_logo.pdf` | PDF | Vetorial | Documentos oficiais |

## Dimensões

- **Largura padrão**: 2000px (alta resolução)
- **Manter proporção**: Sempre
- **Tamanho mínimo**: 50px de largura

## Quando Usar

Este é o logo **padrão** a ser usado em:
- Documentos oficiais
- Apresentações
- Material de divulgação
- Website institucional
- Publicações científicas

## Não Usar Se

- Fundo for muito colorido (use variante apropriada)
- Tamanho for muito pequeno (use versão simplificada)
- For impressão P&B (use versão monocromática)

## Cores

O logo principal contém as cores oficiais do IBS:
- Verde IBS: #228B22
- Terra: #8B5A2B
- Texto: #333333

## Download

Faça download do formato apropriado para seu uso. Sempre consulte o manual de identidade visual antes de usar.
